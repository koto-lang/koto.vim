# koto.vim

Syntax highlighting and indentation support for [Koto][koto] in Vim/Neovim.

## Installation

### [vim-plug][vimplug]

```vim
Plug 'koto-lang/koto.vim'
```

### [Pathogen][pathogen]

```sh
git clone https://github.com/koto-lang/koto.vim.git ~/.vim/bundle/koto.vim
```

[koto]: https://github.com/koto-lang/koto
[vimplug]: https://github.com/junegunn/vim-plug
[pathogen]: https://github.com/tpope/vim-pathogen
